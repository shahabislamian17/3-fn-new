# **App Name**: 3JN Fund Platform

## Core Features:

- Multi-user crowdfunding platform: Develop a web application supporting Equity Share and Royalty/Revenue-Based Profit Share investment models, and that is multi-user.
- AI-powered investment decisioning: Automatically generate 3-year cashflow and P&L projections for investment opportunities using a deterministic formulaic engine. Use an LLM tool to normalize inputs and suggest assumptions.
- KYC/AML compliance: Implement KYC and AML verification processes for investors and project owners, ensuring compliance with regulatory requirements.
- Multi-currency and multi-language support: Enable the platform to support multiple languages and currencies with live exchange rates for global accessibility.
- Admin dashboards and controls: Admin will need panels for managing projects, campaigns, user roles, commission, approval flows, currency exchange rates, compliance and accounting.
- Escrow service integration: Integrate with an escrow service provider to ensure funds are held securely until campaign conditions are met.
- Project health metrics: Monitor a project's overall health metrics in the dashboard such as amount raised vs target and AI suggested optimizations.

## Style Guidelines:

- Primary color: Deep teal (#008080) to convey trust, security, and sophistication suitable for a financial platform.
- Background color: Light gray (#F0F8FF), providing a clean and professional backdrop that allows the primary elements to stand out.
- Accent color: Muted gold (#B8860B), for highlighting important information and calls to action, evoking a sense of value and exclusivity.
- Body font: 'PT Sans' (sans-serif) is suitable for both headlines or body text; pairing simplicity and clarity
- Maintain a clean, well-organized layout across all platforms with clear navigation and intuitive user flows, and consider accessibility.
- Use professional, easily recognizable icons throughout the platform to aid navigation and communicate key concepts at a glance.
- Incorporate subtle transitions and animations to provide feedback and enhance the user experience without being distracting.
